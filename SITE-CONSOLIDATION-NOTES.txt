For a normal lesson update, edit learning-content.js only. It is loaded directly by the website, so no minification step is required.

WHEN EXCEL SESSION 5 IS PUBLISHED
---------------------------------
In js/learning-content.js, find the lesson with:
  id: "excel-5"

Then:
1. Add videoId and videoUrl.
2. Change status from "coming-soon" to "published".
3. Add an assignmentId only if an assignment is available.
4. Add a matching assignment in the assignments section only when ready.

The Courses page, YouTube page, Search, and Assistant will update from that central source.

EDITING WEBSITE CODE
--------------------
After editing CSS or JavaScript source code, regenerate minified assets:
  py -m pip install rcssmin rjsmin
  py build-minified-assets.py

Do not run the build script for normal lesson-data changes in learning-content.js.
