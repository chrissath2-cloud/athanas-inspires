Creating a Visual Document in Microsoft Word

Internet Safety Guide Example

Follow these steps carefully:

  1. Watch Word Session 3 on YouTube first.
  2. Download the image resource ZIP from this page.
  3. Extract the ZIP file on your computer.
  4. Recreate the mini project in Microsoft Word using the preview image as your guide.
  5. Submit your work through WhatsApp when you finish.

This assignment helps you practice pictures, Wrap Text, shapes, text boxes, WordArt, symbols, equations, and clean visual layout.

Watch Word Session 3 before starting the assignment

Your recreated document should include:

  • Clear title and subtitle
  • WordArt slogan: Stay Safe Online
  • One inserted picture using the downloadable image
  • Shapes, text box, and warning message
  • Equation: Strong password = Letters + Numbers + Symbols
  • Quick safety checklist with check and cross symbols
  • Conclusion box at the bottom

Simple grading guide

Excellent: Clear, complete, and very close to the preview.

Good: Mostly correct, with small improvements needed.

Needs Improvement: Good effort, but needs more practice with layout and visual control.
